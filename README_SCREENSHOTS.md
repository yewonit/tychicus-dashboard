# DUGIGO 청년회 관리 시스템 - 화면 캡처 모음

이 프로젝트는 DUGIGO 청년회 관리 시스템의 각 메뉴별 화면들을 시뮬레이션한 HTML 파일들의 모음입니다.

## 📁 생성된 화면 목록

### 1. 전체현황 (대시보드)
- **파일**: `dashboard_screenshot.html`
- **설명**: 청년회 전체 현황을 한눈에 볼 수 있는 대시보드 화면
- **주요 기능**:
  - 통계 카드 (구성원 수, 출석률, 심방 건수, 활성 포럼 수)
  - 차트 섹션 (출석률 트렌드, 국별 현황)
  - 최근 활동 목록
  - 필터링 옵션 (기간, 국, 그룹)

### 2. 구성원 관리
- **메인화면**: `members_management_screenshot.html`
- **상세화면**: `member_detail_screenshot.html`
- **모달화면**: `add_member_modal_screenshot.html`

#### 2.1 메인화면
- **파일**: `members_management_screenshot.html`
- **설명**: 구성원 목록을 관리하는 메인 화면
- **주요 기능**:
  - 구성원 검색 및 필터링
  - 구성원 테이블 (이름, 소속, 연락처, 상태, 최근 활동)
  - 새 구성원 추가 버튼
  - 각 구성원별 작업 버튼 (보기, 수정, 삭제)
  - 페이지네이션

#### 2.2 상세화면
- **파일**: `member_detail_screenshot.html`
- **설명**: 개별 구성원의 상세 정보를 보는 화면
- **주요 기능**:
  - 프로필 카드 (아바타, 이름, 직분, 기본 정보)
  - 상세 정보 (소속, 담당 지역, 출석률 등)
  - 최근 활동 목록
  - 작업 버튼 (수정, 심방 등록)

#### 2.3 모달화면
- **파일**: `add_member_modal_screenshot.html`
- **설명**: 새 구성원을 추가하는 모달 창
- **주요 기능**:
  - 기본 정보 입력 (이름, 생년월일, 연락처, 이메일, 주소)
  - 소속 정보 선택 (국, 그룹, 직분, 담당 지역)
  - 추가 정보 및 메모
  - 프로필 사진 업로드

### 3. 포럼 관리
- **메인화면**: `forum_management_screenshot.html`
- **모달화면**: `create_forum_modal_screenshot.html`

#### 3.1 메인화면
- **파일**: `forum_management_screenshot.html`
- **설명**: 청년회 포럼을 관리하는 메인 화면
- **주요 기능**:
  - 탭 메뉴 (포럼, 기도제목)
  - 포럼 검색 및 필터링
  - 포럼 카드 그리드 (제목, 카테고리, 내용, 참여자 수, 댓글 수)
  - 새 포럼 생성 버튼
  - 각 포럼별 작업 버튼

#### 3.2 모달화면
- **파일**: `create_forum_modal_screenshot.html`
- **설명**: 새 포럼을 생성하는 모달 창
- **주요 기능**:
  - 기본 정보 입력 (제목, 카테고리, 태그)
  - 포럼 내용 작성 (설명)
  - 미리보기 기능
  - 포럼 설정 (공개/비공개, 댓글 검토, 알림 등)
  - 접근 권한 설정

### 4. 심방 관리
- **메인화면**: `visitation_management_screenshot.html`
- **상세화면**: `visitation_detail_screenshot.html`

#### 4.1 메인화면
- **파일**: `visitation_management_screenshot.html`
- **설명**: 청년회 구성원 심방 현황을 관리하는 메인 화면
- **주요 기능**:
  - 통계 개요 (이번 달 심방, 대상자, 진행자, 달성률)
  - 검색 및 필터링 (대상자, 국, 심방 방법, 기간)
  - 심방 테이블 (대상자, 진행자, 날짜, 방법, 내용)
  - 새 심방 등록 버튼
  - 페이지네이션

#### 4.2 상세화면
- **파일**: `visitation_detail_screenshot.html`
- **설명**: 개별 심방의 상세 정보를 보는 화면
- **주요 기능**:
  - 기본 정보 (심방 날짜, 방법, 상태, 작성일시)
  - 대상자 정보 (이름, 소속, 직분, 연락처, 주소)
  - 진행자 정보 (이름, 소속, 직분, 연락처)
  - 심방 내용 (상세 내용, AI 개선 제안, 키워드 분석)
  - 첨부파일 (사진, 보고서 등)

## 🎨 디자인 특징

### 색상 팔레트
- **주요 색상**: #26428b (파란색)
- **보조 색상**: #1e2d7a (진한 파란색)
- **강조 색상**: #ffd700 (골드)
- **배경**: #f8f7f5 ~ #e8e6e1 (그라데이션)

### UI 컴포넌트
- **카드 디자인**: 둥근 모서리, 그림자 효과
- **버튼**: 호버 효과, 그라데이션 배경
- **테이블**: 호버 효과, 상태 배지
- **모달**: 슬라이드 인 애니메이션, 오버레이

### 반응형 디자인
- **사이드바**: 280px 고정 너비
- **그리드 레이아웃**: 자동 조정되는 컬럼
- **모바일 대응**: 태블릿 크기까지 지원

## 🚀 사용 방법

1. 각 HTML 파일을 웹 브라우저에서 열기
2. 파일명에 따라 해당 메뉴의 화면 확인
3. 실제 시스템과 동일한 UI/UX 경험

- Firefox
- Safari
- Edge

- **모던 UI**: 그림자, 그라데이션, 호버 효과
