# 청년회 조직 구조

## 조직 개요
청년회는 교역자를 중심으로 한 계층적 조직 구조로 운영되며, 전체 청년들을 체계적으로 관리하고 섬기는 시스템을 갖춘 교회 부서입니다.

## 조직 구성

### 1. 교역자
- **목사, 전도사**
- **역할**: 청년회 전체를 총괄하고 영적 지도를 담당
- **소속**: 조직 전체를 관할하며 국, 그룹, 순에 직접 소속되지 않음

### 2. 회장단
- **역할**: 교역자를 보필하며 청년회 전체 조직을 기획하는 최고 청년 리더층
- **소속**: 각자 국, 그룹, 순에 소속되어 있음
- **구성**:
  - 회장
  - 부회장
  - 서기
  - 부서기
  - 회계
  - 부회계
  - 총무

### 3. 그룹국 국장단
- **역할**: 청년회 전체 인원을 연령대별로 5개국으로 나누어 각 국을 관리
- **인원 구성**:
  - **국장**: 각 국별 1명씩 총 5명
  - **부국장**: 각 국별 1명씩 총 5명
- **소속**: 각자 담당하는 국, 그룹, 순에 소속되어 있음
- **관할 범위**: 각 국당 3~4개 그룹을 관리

### 4. 그룹장단
- **역할**: 각 그룹국을 3~4개 그룹으로 세분화하여 각 그룹을 관리
- **인원 구성**:
  - **그룹장**: 각 그룹별 1명 (전체 청년회 기준 15~20명)
  - **부그룹장**: 각 그룹별 1명 (전체 청년회 기준 15~20명)
- **소속**: 각자 담당하는 그룹과 순에 소속되어 있음
- **관할 범위**: 각 그룹당 3~4개 순을 관리

### 5. 순장
- **역할**: 각 그룹을 3~4개 순(팀)으로 나누어 각 순을 직접 관리하는 소그룹 리더
- **인원 구성**: 각 순별 1명 (전체 청년회 기준 45~80명)
- **소속**: 각자 담당하는 순에 소속되어 있음
- **관할 범위**: 순원, EBS, 새가족을 포함한 순 전체 구성원 관리

### 6. EBS (새가족 담당자)
- **역할**: 각 순의 새가족을 전담하여 돌보는 청년
- **인원 구성**: 
  - 기본적으로 각 순별 1명
  - 새가족이 많은 경우 순별 최대 2명까지 가능
- **소속**: 각자 담당하는 순에 소속되어 있음
- **담당 업무**: 새가족의 적응과 정착을 위한 전담 케어

### 7. 순원 (기존 청년)
- **역할**: 각 순의 팀원으로 활동하는 기존 청년들
- **인원 구성**: 각 순별 최소 2명 이상
- **소속**: 각자 속한 국, 그룹, 순에 소속
- **구성**:
  - 활발히 참여하는 기존 청년
  - 단기결석자 (일시적 결석)
  - 장기결석자 (지속적 결석이지만 여전히 소속 유지)

### 8. 새가족
- **역할**: 청년회에 새로 등록한 청년들
- **인원 구성**: 각 순별로 존재할 수도, 존재하지 않을 수도 있음
- **소속**: 배정받은 국, 그룹, 순에 소속
- **특징**: EBS의 전담 케어를 받으며 청년회 적응 과정 중

## 조직의 특징

**계층적 구조**: 교역자 → 회장단 → 국장단 → 그룹장단 → 순장 → 순원의 명확한 위계 구조

**체계적 관리**: 전체 청년을 연령대별 국 → 그룹 → 순으로 세분화하여 효율적으로 관리

**새가족 중심**: EBS 시스템을 통해 새가족을 별도로 관리하는 새가족 친화적 구조

**포용적 접근**: 단기/장기 결석자까지 포함하여 모든 청년을 조직 내에서 관리

**규모 산정**: 
- 5개국 × 3~4개그룹 × 3~4개순 = 총 45~80개 순으로 구성
- 각 순별 최소 3명(순장+EBS+순원) 이상으로 전체 135~240명 규모
- 리더 비율: 전체 인원의 약 30~40% (국장단, 그룹장단, 순장, EBS 포함)

**소속 원칙**: 교역자를 제외한 모든 구성원은 반드시 특정 국, 그룹, 순에 소속되며, 리더 역할을 맡은 경우에도 자신의 소속 순에서 순원으로서의 역할을 동시에 수행

## 예배 및 모임

### 정기 예배
- **대예배**: 매주 일요일 오전 (주일 청년예배 전)
- **주일청년예배**: 매주 일요일 오전 (대예배 후)
- **수요제자기도회**: 매주 수요일 저녁
- **두란노사역자모임**: 매주 금요일 저녁

### 특별 모임
- **심방**: 구성원 돌봄 및 영적 상담
- **지역모임**: 지역별 소그룹 모임
- **포럼**: 청년회 전체 모임 및 공지사항
